# CharaSearch

ブラウザ上で画像内のキャラクターと作品名を右クリックから即座に特定できるChrome拡張機能＋C#サーバーです。

## 概要
* **目的:** Web閲覧中に気になるキャラクターの元ネタを素早く検索する。
* **仕組み:**
    1. ブラウザ拡張機能で対象画像をBase64形式で取得。
    2. C#で作成したローカルサーバーへ画像を送信。
    3. **SauceNAO API** を介して画像検索を行い、元画像や出典元を特定。
    4. ブラウザに結果をポップアップ表示。

## 技術スタック
* **Frontend:** JavaScript (Chrome Extension API)
* **Backend:** C# (.NET 8, ASP.NET Core)
* **Search Engine:** [SauceNAO API](https://saucenao.com/)

## セットアップ手順

### 1. サーバーの準備
1. プロジェクトフォルダに移動します。
2. [SauceNAO API Account](https://saucenao.com/user.php?page=search-api) で取得したAPIキーを `Program.cs` 内の `apiKey` に設定してください。
3. サーバーを起動します：
   ```bash
   dotnet run

現在、本プロジェクトは**実験段階**です。

## 制限事項（既知の課題）
現在、以下の理由によりキャラクター判別の精度が限定的です：
* **画像一致検索の限界:** SauceNAO APIは完全一致や類似画像検索を行う性質上、トリミングや加工が激しい画像、あるいはDBに登録されていないイラストについては特定が困難です。
* **実験的実装:** 現在はPoC（概念実証）段階であり、エラーハンドリングや検索結果のパース処理は最小限の実装となっています。
* **今後の課題:** 将来的には、より高精度なAIモデル（Vision API等）の導入や、検索結果の複数候補表示など、精度向上のための修正を検討しています。
